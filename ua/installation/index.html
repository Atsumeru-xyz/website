    # Підготовка

    Гайд актуальний для сервера версії 1.0.0+

    Встановлення Atsumeru дуже просте і складається з пари кроків

    # Основні способи встановлення

    • Використання Java і Jar файлу сервера

    # Додаткові необхідні додатки

    Так як сервер не надає WebUI для управління сервером, додатково необхідно завантажити Atsumeru Manager (opens new window) через який і відбуватиметься управління

    Додаток не потребує встановлення, але вимагає наявності встановленого Microsoft Visual C++ 2010 (x64) (opens new window) і Microsoft Visual C++ 2015 (x64) (opens new window) для Windows систем

    Додаток працює тільки під Windows x64, Linux x64 та Mac OS x64 (робота на M1/M2 не тестувалася)
