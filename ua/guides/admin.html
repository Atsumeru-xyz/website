# Адміністрування

У цьому розділі розташовано опис функцій, які доступні тільки Адміністраторам

# Перевірити, чи скачано контент за посиланнями

Дає змогу перевірити, чи скачаний та імпортований контент, у якого в Метаданих вказані надані посилання

Інтерфейс перевірки посилань

# Очистити кеш сервера Небезпечно

Змушує сервер повністю очистити кеш обкладинок і запускає сервіс кешування

Тривалість цієї процедури прямо пропорційна кількості імпортованих файлів

# Створити унікальні хеші архівів Небезпечно

Для кожного Архіву і Серії створює унікальні статичні хеші та записує їх у book_info.json метадані

Тривалість цієї процедури прямо пропорційна кількості імпортованих файлів

# Записати всі метадані в архіви Небезпечно

Якщо для прискорення редагування метаданих запис здійснювався тільки в базу даних, можливо запустити процес запису всіх метаданих із бази в book_info.json файли

Просунуті користувачі, знайомі з SQLite (opens new window)базами даних, можуть масово змінювати дані Архівів безпосередньо в базі та використовувати цю функцію для запису змін у файли

Тривалість цієї процедури прямо пропорційна кількості імпортованих файлів
