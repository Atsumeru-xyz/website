    # Additional required applications

    Since the server does not provide a WebUI to manage the server, you must additionally download Atsumeru Manager (opens new window) , through which management will take place

    The application does not require installation, but it does require you to have Microsoft Visual C++ 2010 (x64) (opens new window) and Microsoft Visual C++ 2015 (x64) (opens new window) installed

    Application only works on Windows x64 and Linux x64
