# Samba (SMB)

If your files are stored outside of a server, you can use Samba (SMB) (opens new window) to connect this storage to server

Use instructions for your operating system to create and connect Samba (SMB) (opens new window) to server

Once storage is connected, directories from it can be Added to importer as local directories. Files added from this storage can be modified, so editing Metadata will work as usual

Under Windows it is recommended to mount the repositories as virtual drive letters, though mounting as a network location will also work
