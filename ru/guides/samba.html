# Samba (SMB)

Если ваши файлы хранятся вне сервера, то можно использовать Samba (SMB) (opens new window) для подключения данного хранилища к серверу

Используйте инструкцию для вашей операционной системы по созданию и подключению Samba (SMB) (opens new window) к серверу

Как только хранилище будет подключено, директории из него можно Добавлять в импортер как при работе с локальными директориями. Файлы добавленные с такого хранилища можно модифицировать, поэтому редактирование Метаданных будет работать как обычно

Под Windows рекомендуется монтировать хранилища в виртуальные буквенные диски, хотя подключение в виде сетевого расположения также будет работать
