    # Подготовка

    Гайд актуален для сервера версии 1.0.0+

    Установка Atsumeru очень простая и состоит из пары шагов

    # Основные способы установки

    • Использование Java и Jar файла сервера

    # Дополнительные необходимые приложения

    Так как сервер не предоставляет WebUI для управления сервером, дополнительно необходимо скачать Atsumeru Manager (opens new window) через который и будет происходить управление

    Приложение не требует установки, но требует наличие установленного Microsoft Visual C++ 2010 (x64) (opens new window) и Microsoft Visual C++ 2015 (x64) (opens new window) для Windows систем

    Приложение работает только под Windows x64, Linux x64 и Mac OS x64 (работа на M1/M2 не тестировалась)
